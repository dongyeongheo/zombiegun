# ZOMBIEGUN

Three.js 기반 1인칭 좀비 서바이벌 시뮬레이터. 안전구역에서 스폰해서 오픈필드의 좀비를 사냥하고, 코인을 모아 무기와 방어구를 강화하는 게임.

## 플레이 방법

| 조작 | 키 |
|---|---|
| 이동 | W A S D |
| 시점 | 마우스 |
| 공격/발사 | 좌클릭 |
| 달리기 | Shift |
| 점프 | Space |
| 인벤토리 | Tab / I |
| 상점 / 상자 열기 | E |
| 무기 전환 | 1 ~ 8 / 마우스 휠 |

## 게임 요소

- **안전구역**: 철망 울타리로 보호되는 스폰 지점. 텐트, 상점, NPC 군인 배치
- **NPC 군인**: 정문 경비 4명 + 2인 1조 순찰 8명. 좀비가 접근하면 텐트에서 지원 병력 10명(저격수 2 + 소총수 8) 출동
- **좀비 6종**: 워커 / 워커대장 / 러너 / 알파 / 브루저 / 슈퍼좀비 — 대장급은 무리를 인솔하고, 무리 중 하나가 맞으면 전체가 어그로
- **무기 8종**: 도끼(기본) → 활(화살 투사체) → 대검 → .22구경 → 더블배럴샷건 → 돌격소총 → RPG(광역폭발) → 대물저격총(관통)
- **방어구**: 일반 / 쇠 / 현대 군 등급의 갑옷·투구, 감소율 합산 (최대 40%)
- **코인**: 바닥 코인 + 좀비 처치 시 물리 드랍 (자석 흡수)
- **보물상자**: 맵 곳곳에 랜덤 배치, 열면 코인 1000~3000개 (새로고침 시 위치 리셋)
- **핫바**: 마인크래프트식 하단 슬롯, 인벤토리 열고 슬롯 클릭 2번으로 배치 변경
- 진행상황은 localStorage에 자동 저장

## 로컬 실행

정적 사이트라서 아무 HTTP 서버로 실행 가능:

```bash
npx serve .
# 또는
python -m http.server 8000
```

브라우저에서 `http://localhost:3000` (serve) 또는 `http://localhost:8000` 접속.

> ES 모듈을 쓰기 때문에 `file://`로 직접 열면 안 되고 반드시 HTTP 서버로 열어야 함.

## Vercel 배포

1. [vercel.com](https://vercel.com) 로그인 → **Add New Project**
2. 이 GitHub 레포(`zombiegun`) import
3. Framework Preset: **Other**, 빌드 설정 없이 그대로 Deploy
4. 끝 — 정적 파일이라 빌드 과정 없음

또는 CLI로:

```bash
npx vercel --prod
```

## 기술 스택

- [Three.js](https://threejs.org/) r160 (CDN importmap, 번들러 없음)
- 순수 ES 모듈 — 빌드 과정 없는 정적 사이트
- 텍스처는 전부 Canvas API로 런타임 생성 (외부 에셋 0개)
